## 快速开始

### 1. 添加新示例
1. 在 `src/data/examples.ts` 中添加示例配置
2. 创建示例目录和文件
3. 实现示例代码和文档
4. 更新开发状态

### 2. 开发规范
- 遵循渐进式的示例展示方式
- 每个示例必须包含完整代码
- 添加详细的注释和解析

### 3. 文档维护
- 及时更新开发状态
- 记录重要决策和变更
- 保持文档的同步更新
